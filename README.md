# Hildebrando e Leite

Site estatico preparado para deploy no Cloudflare Pages.

## Deploy no Cloudflare Pages

- Framework preset: `None`
- Build command: deixe vazio
- Build output directory: `/`
- Root directory: `/`

O arquivo principal do site e `index.html`.

## Observacoes

Este projeto foi criado a partir de uma exportacao estatica de um site WordPress/Elementor. Algumas fontes, imagens, estilos e scripts ainda sao carregados de `https://hildebrandoleite.com.br`. Para deixar o site 100% independente no futuro, esses assets devem ser baixados e os caminhos devem ser convertidos para arquivos locais.
